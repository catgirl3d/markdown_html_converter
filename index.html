Markdown → HTML

Конвертор с поддержкой разметки и переносов строк

Новые функции:

  • Кнопки для копирования Markdown, вставки, очистки, копирования HTML-вывода, копирования полного HTML-документа и скачивания HTML.
  • Кнопка "Копировать" для каждого блока кода в HTML-выводе.
  • Сочетания клавиш:
    • Ctrl/Cmd + V: Вставить Markdown
    • Ctrl/Cmd + C (при фокусе на HTML-выводе): Копировать HTML-вывод
    • Ctrl/Cmd + Shift + C: Копировать полный HTML-документ
